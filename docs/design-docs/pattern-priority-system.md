# Pattern Priority System

## 결정 배경

AI 냄새 패턴은 우선순위에 따라 적용 순서가 결정된다.
텍스트 수정 시 패턴 간 충돌을 방지하기 위해 도입.

## 우선순위 레벨

| 레벨 | 의미 | 예시 |
|:--|:--|:--|
| `p0` | 반드시 제거해야 하는 강력한 AI 신호 | "결론부터 말씀드리면" → 삭제 |
| `p1` | 개선이 필요한 AI 패턴 | "정말 최고의 선택이었어요" → "좋은 선택이었어요" |
| `p2` | 선택적 개선 (기본값) | 일반적인 hedging 표현 |

## 적용 규칙

1. `sortPatternsByPriority()`로 p0 → p1 → p2 순서로 정렬
2. 순서대로 한 패턴씩 적용 (이전 패턴 결과에 다음 패턴 적용)
3. 변경이 발생한 경우만 Fix 기록에 추가

## 대안 검토

- **병렬 적용**: 패턴 간 충돌 위험 → 기각
- **우선순위 없는 순차**: 중요도 구분 불가 → 기각
- **3단계 우선순위**: 현재 채택. 확장 필요 시 레벨 추가 가능
